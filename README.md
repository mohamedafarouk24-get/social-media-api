# Social Media API

RESTful Social Media API built with:

- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- JWT Authentication

## Features

- User Registration
- User Login
- JWT Authentication
- Create Post
- Get All Posts
- Get Post By ID
- Get User Posts
- Update Post
- Delete Post
- Like Post
- Unlike Post
- Add Comment
- Delete Comment

Environment Variables
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

## Installation

```bash
npm install
npm run dev
```
